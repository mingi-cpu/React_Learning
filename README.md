# React Learning Repository

## 📌 소개

이 레포지토리는 React 강의를 기반으로 한 **프로젝트 따라 만들기**, **기능 수정/확장**, 그리고 **복습용·연습용 프로젝트 제작**을 기록하는 공간입니다.
단순 따라치기에 그치지 않고, 기존 기능을 개선하거나 새로운 기능을 추가하며 이해도를 높이는 것을 목표로 합니다.

---


## 🎯 목표

* React 기본 개념 확실히 이해하기
* 컴포넌트 설계 능력 키우기
* 간단한 프로젝트를 혼자 구현할 수 있는 수준 도달

---

## 🛠 사용 기술

* JavaScript (ES6+)
* React

---

## 📖 학습 방식

* 강의 프로젝트를 따라 구현
* 구현한 코드에 기능 추가 및 개선
* 동일 기능을 스스로 다시 만들어보며 복습
* 작은 실험/연습 프로젝트로 개념 확인

---


## 💡 참고

이 레포는 학습 기록용이며,
완성도 있는 프로젝트는 별도의 레포지토리로 분리하여 관리할 예정입니다.
